# Deposit Calculator REST API

REST API для управления пользователями и депозитами с функциональностью расчета ежемесячной капитализации процентов

[Техническое задание](https://github.com/user-attachments/files/23315530/tz.pdf)

## Функционал

* Управление пользователями: создание, получение, обновление, удаление
* Управление депозитами: создание, получение, обновление, удаление
* Расчет депозита с ежемесячной капитализацией процентов
* Валидация входных данных и обработка ошибок
